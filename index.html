Css Tips and Tricks

Check out these sweet css snippets! I made them myself.

Reset

Browsers like to apple their own stylesheets. Meaning your code can look different on different browsers. This snippet wikk lightly reset that.

  *{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }

Flex stuff

Want all your items to line up nicely? try flexbox!

    main{
        background: chartreuse;
        min-height: 70vh;
        padding: 1%;
        display: flex;
        flex-wrap: wrap;
        justify-content: space-around;
    }
